傳感器的客製化：

Rotor 編碼器

FRC 隊常用 CTRE CANCoder 來量 rotor 的角度。它是一個很好的傳感器，但價錢有點高。如果想找比較便宜的選項， 可以考慮用 Thrifty Absolute Magnetic Encoder。 這個 編碼器和其他的CANCoders使用一樣的安裝配件，但價錢低很多。
